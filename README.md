Este é um App desenvolvido para o desafio da GO.k. Ele busca as informações da api informada e as exibe na tela.


Stack utilizada:

- Kotlin
- Android DataBinding
- AndroidX
- Retrofit2.0
- RxJava
- MVVM
- LiveData
- Glide
- Android ViewModel
- Android JetPack
- Android Arch Lifecycle
- Navigation
